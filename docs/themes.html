Favorites and ToDo - Themes

Themes are implemented in a very basic way. The colors that are used are defined as CSS variables in a *-theme CSS class. Currently there is a dark-theme and and a light-theme.
The theme can be set by using !theme name where name is the CSS class name of the theme or auto (the default) in which case the browser's dark/light-mode preference setting is used.

Examples: 

!theme dark-theme

!theme light-theme

!theme auto

Background

A background image (wallpaper) can be specified with a relative or absolute URL or with anything else that is acceptable by the CSS background-image property.

Background image with relative URL: 

!wallpapar images/my-background.jpg
Random background image (with absolute URL): 
!wallpaper https://lipsum.app/random/1920x1080
Random background image with no translucency: 
!wallpaper https://lipsum.app/random/1920x1080 100%
Rainbow: 
!wallpaper linear-gradient(120deg, red,orange,yellow,green,blue,indigo,violet,red,orange,yellow,green,blue,indigo,violet)

Colors

A theme is defined by the following variables:

  • --foreground, --background: Default foreground and background colors.
  • --accent-color: Color for titles, labels and icons.
  • --block-foreground, --block-background, --block-border: Foreground, background and border colors for Block level items.
  • --selection-foreground, --selection-background: Foreground and background color for selected text.
  • --code-foreground, --code-background: Foreground and background color for <code> and <pre> fragments.
  • --link-color: Foreground color of hyperlinks.
  • --disabled-color: Foreground color for disabled items.
  • --error-foreground, --error-background, --error-border: Foreground, background and border color for search box in case of unrecognized input.
These variables can be modified in a !style block to modify or create a new style.

A theme can be modified like this: 

!style
.dark-theme {
    --accent-color: #CC6600;
}
!endstyle

It is also possible to override any element: 

!style
body {
	filter: sepia(0.7);
}
!endstyle
