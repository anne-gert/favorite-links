Favorites and ToDo - Drag-and-Drop

Drag-and-Drop from other applications is supported for the following types:

  • Links, e.g. from a browser's address bar.
  • Text that includes a link on every non-empty line. Such lines may also contain a description. A text may contain multiple links.
  • Text without links is included as a !text...!endtext block. Such text may include HTML.
The dragged item is inserted with respect to the item that is closest to the drop location (the target). How it is inserted, depends on the type (link or text) of item that is being added as well as on the type of the target item.

If the target is a link and if the item is dropped above or to the left, it is inserted on the same level, before the target. If it is dropped below or to the right, it is inserted on the same level, after the target. If it is dropped on top of the target link, it is added as a sub-item of that link.
See the images below for clarification.

If the target is a text block and if the item is dropped above or to the left, it is inserted on the same level, before the target. If it is dropped below or to the right, it is inserted on the same level, after the target. If it is dropped on top of the target, it is also added before or after the target (because a text block cannot have sub-items), depending on whether is was dropped on the top half or the bottom half of the target text block.
See the images below for clarification.
