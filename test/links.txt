Test Cases for Icons

!text
<p>
Test cases for handling of icons and icon definitions.
</p>
<p>
Different orders of definitions for normal definitions and different
loop-lengths for refloop cases are shown.
</p>
!endtext

    Char vs. HTML
        Char
            icon=char:X Short Char icon
            icon=char:ICONTEXT Long Char icon
        HTML
            icon=html:X Short HTML icon
            icon=html:ICONTEXT Long HTML icon

    Definition
        Normal
            !icondef ic-ok char:X
            icon=ic-ok Normal icon (should display *'X'*)
        Undefined
            icon=ic-undefined Undefined icon (should display *'!ND!'*)

    Re-definition
        !icondef ic-redefine char:A
        icon=ic-redefine First use of icon (should display *'B'*)
        !icondef ic-redefine char:B
        icon=ic-redefine Second use of icon (should display *'B'*)

    Definition Loop
        !icondef ic-refloop-1a ic-refloop-1b
        !icondef ic-refloop-1b ic-refloop-1a
        icon=ic-refloop-1a Circularly defined icon (length 2) (should display *'!ND!'*)

        !icondef ic-refloop-2a ic-refloop-2b
        !icondef ic-refloop-2b ic-refloop-2c
        !icondef ic-refloop-2c ic-refloop-2d
        !icondef ic-refloop-2d ic-refloop-2e
        icon=ic-refloop-2a Circularly defined icon (lengt 4) (should display *'!ND!'*)

        !icondef ic-refloop-3a ic-refloop-3b
        !icondef ic-refloop-3b ic-refloop-2a
        icon=ic-refloop-3a Circularly defined icon (lengt 6) (should display *'!ND!'*)

Test Cases for Checkboxes

!text
<p>
Test cases for handling of checkboxes and checkbox definitions.
</p>
<p>
Default, undefined and re-defined checkboxes are tested.
Also a page with different ways of Custom Checkbox definitions is linked.
</p>
!endtext

    Definition
        Normal
            [] Default checkbox (the normal case)
        Last Definition
            []: Last defined checkbox is none (should display default)
            !checkbox cb-last-def-1 A B C
            []: Last defined checkbox is defined (should display *A/B/C*)
            !checkbox cb-last-def-2 X Y Z
            []: Last defined checkbox is other defined (should display *X/Y/Z*)
            !checkbox cb-last-def-2 I char:II char:III
            []: Last defined checkbox (is re-defined) (should display *I/II/III*)
        Undefined
            []:cb-undefined Undefined checkbox (should display default checkbox)
        Custom Checkboxes icon=char:T url=?links-override=test/checkbox-links.txt

        With Defaults
            Test with defined default checkbox icon=char:T url=?links-override=test/checkbox-define-default.txt

    No Options
        !checkbox cb-empty-1
        []:cb-empty-1 Empty checkbox definition (should display *!EE!*)
        !checkbox cb-empty-2 !back
        []:cb-empty-2 Empty checkbox definition (should display *!EE!*)
        !checkbox cb-empty-3 cb-empty-2 !front
        []:cb-empty-3 Empty checkbox extension (should display *!EE!*)

    Re-definition
        []:cb-redefine-1 Use before single definition (should display *I/II/III*)
        !checkbox cb-redefine-1 I char:II char:III
        []:cb-redefine-1 Use after single definition (should display *I/II/III*)

        []:cb-redefine-2 Use before definition (should display final value *X/Y/Z*)
        !checkbox cb-redefine-2 A B C
        []:cb-redefine-2 Use after first definition (should display *A/B/C*)
        !checkbox cb-redefine-2 1 2 3
        []:cb-redefine-2 Use after second definition (should display *1/2/3*)
        !checkbox cb-redefine-2 X Y Z

    Extending
        Simple
            !checkbox cb-extend-1a 1 2
            !checkbox cb-extend-1b A cb-extend-1a B
            []:cb-extend-1a Custom Checkbox 1 (base) (should display *1/2*)
            []:cb-extend-1b Custom Checkbox 2 (extend) (should display *A/1/2/B*)
            !checkbox cb-extend-1b X Y
            []:cb-extend-1b Custom Checkbox 2 (redefine) (should display *X/Y*)
        Override Place
            !checkbox cb-extend-2a A B C
            !checkbox cb-extend-2b cb-extend-2a !back
            !checkbox cb-extend-2c cb-extend-2b !front
            []:cb-extend-2a Default place (should display *A/B/C* at *front*)
            []:cb-extend-2b Default place (should display *A/B/C* at *back*)
            []:cb-extend-2c Default place (should display *A/B/C* at *front*)
        Use Before Definition
            []:cb-extend-3-ABC Use/extend before definition (should display *A/B/C*)
            // 1) 3-ABC not yet defined -> use last (and only)
            !checkbox cb-extend-3-ABC A cb-extend-3-BC
            // 2) 3-BC not yet defined -> Use last
            !checkbox cb-extend-3-B B
            !checkbox cb-extend-3-BC X
            !checkbox cb-extend-3-BC cb-extend-3-B C
            // 3) 3-B already defined -> use most recent
            !checkbox cb-extend-3-B Y

    Definition Loop
        !checkbox cb-refloop-1 A cb-refloop-1 B
        []:cb-refloop-1 Custom Checkbox 3 (circular, length=1) (should display *!RL!*)
        !checkbox cb-refloop-2 A cb-refloop-3 B
        !checkbox cb-refloop-3 1 cb-refloop-2 2
        []:cb-refloop-2 Custom Checkbox 5 (circular, length=2) (should display *!RL!*)
        !checkbox cb-refloop-4 X cb-refloop-3 Y
        []:cb-refloop-3 Custom Checkbox 6 (circular, length=3) (should display *!RL!*)
        !checkbox cb-refloop-5 I char:II char:III
        []:cb-refloop-5 Custom Checkbox 7 (not yet circular) (should display *I/II/III*)
        !checkbox cb-refloop-5 cb-refloop-1
        []:cb-refloop-5 Custom Checkbox 7 (now circular) (should display *!RL!*)
